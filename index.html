Add and Remove

Add or remove as many divs as you want! Simply press the "Add Div" or "Delete Div" button.

Change Positioning

Adjust Flex-Direction, Justify-Content, Align-Items, Align-Content, and Flex-Wrap to change the positioning of the divs!

Change Size

Adjust div height and width using "Change Height" and the "Change Width" tabs!

Flex Layout Generator 1.0 was created by me, Gerardo Balduvin, as a first project to get familiar with coding. Not only did I think this would be a great first project, but it would also be a website designed to help other new web developers on their journey to understanding CSS, and also landing that first interview. Whether it be your own personal portfolio, or something as simple as a pamphlet, Flex Layout Generator is here to help!
